# Claim Prompt

청구항 품질을 별도로 개선할 때 사용하는 프롬프트입니다.

```text
다음 발명 아이디어와 명세서 초안을 바탕으로 한국어 특허 청구항 4개를 작성해줘.

발명 아이디어:
{idea}

발명의 명칭:
{title}

발명의 구성:
{configuration}

규칙:
- JSON 배열만 출력한다.
- 총 4개의 문자열을 출력한다.
- 1번째 청구항은 독립항이다.
- 2~4번째 청구항은 "제1항에 있어서,"로 시작하는 종속항이다.
- "포함하는", "특징으로 하는" 표현을 사용한다.
- 실제 법률 효력이 있는 청구항이라고 주장하지 않는다.
- 너무 웃긴 표현 대신 기술적으로 진지한 표현을 사용한다.

좋은 문체 예시:
[
  "생체 가스 배출 압력을 포집하는 포집부와, 상기 포집부로부터 유입된 압력을 저장하는 압력 안정화 챔버와, 상기 압력을 차량 추진력으로 변환하는 추진 노즐을 포함하는 친환경 추진 차량.",
  "제1항에 있어서, 상기 포집부는 차량의 착석부 하부에 배치되는 것을 특징으로 하는 친환경 추진 차량.",
  "제1항에 있어서, 상기 압력 안정화 챔버는 역류 방지 밸브를 포함하는 것을 특징으로 하는 친환경 추진 차량.",
  "제1항에 있어서, 상기 추진 노즐은 악취 저감 필터를 통과한 가스를 배출하는 것을 특징으로 하는 친환경 추진 차량."
]
```
