# Project Brief

## 프로젝트 한 줄 정의

사용자의 황당한 아이디어를 웹 검색, RAG, 작성 에이전트로 분해하고 재조립해 실제 특허 명세서처럼 보이는 문서로 변환하는 AI 에이전트 서비스입니다.

## 서비스 한 줄 정의

내 머릿속의 말도 안 되는 상상을 진짜 특허처럼 보이는 명세서로 뽑아주는 서비스입니다.

## 문제 정의

창의적이고 엉뚱한 상상을 전문성 있는 형식으로 바꿔주는 쉬운 도구가 부족합니다. 특허 명세서처럼 보이려면 "기존 기술과 달리, 본 발명은..." 같은 문체와 청구항 구조가 필요한데, 일반 사용자는 이 형식을 직접 쓰기 어렵습니다.

## 핵심 가치

사용자의 황당한 아이디어를 "진짜 특허 같다"고 느끼게 만드는 그럴듯함입니다.

## 타깃 사용자

- SF, 판타지 크리에이터: 웹소설, 게임 시나리오, 세계관 설정집 보강
- 밈, 유머 SNS 사용자: 친구 선물, 공유용 이미지/PDF, 재미 콘텐츠
- 브레인스토밍 단계의 기획자: 극단적인 아이디어를 정형 문서로 시각화

## 데모 성공 기준

발표장에서 사용자가 한 문장 아이디어를 입력했을 때, 30~60초 안에 특허 명세서처럼 보이는 결과 화면이 나타나야 합니다. 최소한 발명의 명칭, 기술분야, 배경기술, 해결하려는 과제, 발명의 구성, 청구항, 요약이 모두 채워져야 합니다.

## 법적 디스클레이머

이 서비스는 엔터테인먼트 및 창작 보조 목적입니다. 실제 특허 출원, 신규성 판단, 진보성 판단, 법률 자문을 제공하지 않습니다.
