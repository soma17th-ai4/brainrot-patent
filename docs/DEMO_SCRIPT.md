# Demo Script

## 발표 한 줄 소개

Brainrot Patent는 말도 안 되는 상상을 실제 특허 명세서처럼 보이는 문서로 바꿔주는 AI 에이전트 서비스입니다.

## 3분 데모 흐름

1. 문제 제기
   - "재밌는 아이디어는 많은데, 전문적인 문서 형식으로 바꾸는 건 어렵습니다."
   - "특히 특허 문서는 문체와 구조가 특이해서 일반 사용자가 흉내 내기 어렵습니다."

2. 서비스 소개
   - "저희는 황당한 아이디어를 특허 명세서 형식으로 변환하는 에이전트를 만들었습니다."
   - "실제 특허 자문은 아니고, 창작과 유머 목적의 데모 서비스입니다."

3. 입력 시연
   - 추천 입력 1: `방구로 가는 자동차`
   - 추천 입력 2: `기억을 USB처럼 꽂아서 옮기는 헬멧`
   - 추천 입력 3: `잠을 대신 자주는 베개`

4. 생성 과정 설명
   - "입력 문장을 기술 용어로 바꾸고, 검색 근거를 참고한 뒤, 특허 명세서 7개 섹션으로 재구성합니다."

5. 결과 설명
   - 발명의 명칭
   - 배경기술
   - 발명의 구성
   - 청구항
   - 요약

6. 마무리
   - "핵심은 헛소리와 진지한 특허 문체 사이의 간극입니다."
   - "이 간극이 창작자에게는 설정집이 되고, SNS 사용자에게는 공유 가능한 콘텐츠가 됩니다."

## 실패 대비 멘트

API가 느릴 때:

```text
외부 LLM과 검색 API를 호출하기 때문에 네트워크 상황에 따라 시간이 걸릴 수 있습니다. 데모 안정성을 위해 같은 입력의 사전 생성 결과도 준비해두었습니다.
```

검색 API가 실패할 때:

```text
검색 근거 수집이 실패하면 근거 없는 기본 명세서 생성으로 fallback되도록 설계했습니다.
```

PDF가 실패할 때:

```text
PDF 렌더링은 최종 출력 옵션이고, 핵심 기능은 특허 명세서 구조의 생성입니다. 현재 화면의 미리보기 결과가 PDF 템플릿에 들어가는 원본 데이터입니다.
```
